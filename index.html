Sunday Weekly Audit
Full account check
▼
Morning — 9:00–11:00
  • Open all 3 ad accounts in Meta Ads Manager. For every active ad set, record: reach, frequency, results, CPL, spend, and CTR. Compare to last week.
  • Flag every ad set where: frequency exceeded 2.5 / CPL exceeded kill-switch / results declined more than 20% week-on-week.
  • Sync the dashboard app with the latest data (use the date range selector for the current month).
Midday — 11:00–13:00
  • Immediate actions based on current data:
  • Dr. Ihab Pause Libya/Algeria ad set now (freq. 3.26, CPL 30.99 EGP). Do not reactivate until new creative is ready.
  • Top Care Pause Teeth mssg2 (60.59 EGP CPL, 2 results only).
  • Top Care Scale Lasir budget by 40% — it is performing at 1.12 EGP CPL. Do this in one step (not gradual), then monitor for 48hrs.
Afternoon — 14:00–17:00
  • Review audience overlap across all active ad sets using Audience Overlap tool — check if Eye World and Top Care campaigns are competing for the same audience pool.
  • Prepare the week's optimization plan: which ad sets will get new creatives (from Asmaa + Bakr), and on what day will you launch them.
  • Update CPL targets and kill-switch values in the dashboard app based on current performance.

▼

  • Set automated rules in Meta: pause any ad if CPL exceeds kill-switch target for 3 consecutive days.
